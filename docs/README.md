# 치킨 미션

## 기능 목록

### 도메인 기능
- [X] 주문 기능
 - 1개 메뉴의 최대 주문 개수는 99개
- [X] 테이블의 주문 정보를 조회하는 기능
- [X] 결제 기능
  - 치킨 종류 메뉴의 수량 합이 10개가 넘는 경우, 10,000원씩 할인  
    e.g. 10개는 10,000원 할인, 20개는 20,000원 할인
  - 현금 결제는 5% 할인

### 입력 기능
- [X] 기능 입력 기능
- [X] 테이블 번호 입력 기능
- [X] 메뉴 입력 기능
- [X] 결제 방식 입력 기능

### 출력 기능
- [X] 메인화면 출력 기능
- [X] 테이블 목록 출력 기능
- [X] 메뉴 출력 기능
- [X] 주문 내역 출력 기능
- [X] 최종 결제 금액 출력 기능
